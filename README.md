# 🚀 Gestor de Tarefas Pro (Fullstack Spring Boot)

Sistema completo de gestão de tarefas desenvolvido com Java e Spring Boot, apresentando um Dashboard dinâmico e relatórios profissionais.

## 🛠️ Tecnologias Utilizadas

- **Backend:** Java 17, Spring Boot 3, Spring Security, Spring Data JPA.
- **Banco de Dados:** PostgreSQL.
- **Frontend:** Thymeleaf, Bootstrap 5, Chart.js.
- **Relatórios:** Apache POI (Excel) e iText (PDF).

## ✨ Funcionalidades

- [x] Cadastro e gerenciamento de tarefas via interface Web.
- [x] Dashboard dinâmico com gráfico de rosca (Status das Tarefas).
- [x] Autenticação de usuários com Spring Security.
- [x] Exportação de dados para PDF e Planilhas Excel.
- [x] Design Responsivo (Funciona no celular).
